변수같은 경우 2일차에 배울 것이므로 짧게만 배운다.

int < 정수형 변수 그리고 변수 이름

변수 이름 규칙:
    1. 키워드와 같게 짓지 않는다. 키워드는 이미 저장돼있는 단어라고 한다. 예약어라고도 한다.
        - auto, double,int , struct, if, break, else, long ,switch 같은 것
    2. 숫자로 시작할 수 없다.
    3. 특수문자는 기호 문자만 이용 가능
    4. 숫자는 사용 가능하나 처음에는 올 수 없음.

좋은 변수는 그럼 뭘까? => 밑줄 방식, 단어의 첫번째 글자를 대문자.


형식 지정자: printf나 scanf 값을 출력하거나 입력 받을 때 형식을 지정하는 것.
%d 10진수 정수 형식
%f 실수 형식 (float) 형식 
%lf 실수 중 double 형식
%c 문자 형식
%s 문자 형식

형식은 꼭 지켜줘야함.

scanf() < 같은 경우는 "%d" &x < 값을 저장할 변수  : 변수를 저장할 형식 andpercent변수

보통은 저장할 변수와 형식을 꼭 지켜줌
