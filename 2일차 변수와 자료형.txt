변수는 프로그램에서 일시적으로 데이터를 저장하는 공간

변수 필요이유는 데이터가 입력되면 어딘가에 저장해야만 다음에 사용할 수 있기 때문이다.

변수와 대립되는 개념이 상수이다.

변수 : 저장된 값이 변경이 가능하다.

상수 : 저장된 값이 변경이 불가능하다.

변수는 
자료형 변수이름;(세미콜론) 으로 생성한다.

자료형 :  변수가 저장할 데이터의 형태
정수, 실수, 문자  이렇게 크게 3종류

정수 : short, int, long, long long 

실수 : float, double, long double

문자 : char

하나의 변수가 담을 수 있는 공간의 크기 별로 나열된 것이다.

그럼 각각의 크기를 알고 싶을 때는 어떻게 하냐?
sizeof 라는 함수를 이용하면 된다. sizeof는 데이터의 크기를 바이트 단위로 반환한다.

int 4
char 1
short 2
long 4
long long 8
float 4
double 8
long double 8


보통 외울 때 int 는 약 21억
long long 42억 이라고 외우면 편하다.

오버플로우 : 변수가 나타낼 수 있는 범위를 넘어서 변수에 저장하려고 할 때 발생
값이 넘어가게 되며 -음수가 된다.


변수 작명 규칙:
    1. 키워드와 같게 짓지 않는다. 키워드는 이미 저장돼있는 단어라고 한다. 예약어라고도 한다.
        - auto, double,int , struct, if, break, else, long ,switch 같은 것
    2. 숫자로 시작할 수 없다.
    3. 특수문자는 기호 문자만 이용 가능
    4. 숫자는 사용 가능하나 처음에는 올 수 없음.



형식 지정자 
"%d" 10진수 정수
"%ld" 10진수 정수 long long 더 큰 범위를 받는 형식
"%f" float(실수)
"%lf" double(큰 실수)
"%s" 문자열
"%c" 문자 하나


값을 초기화할 때 8진수, 16진수로 넣어줄 수 있다.

예를 들어)
int x = 10  이런식으로 넣게되면 정수 그대로 넣었을 시 10진수로 넣는 것이고
int y = 010 이런식으로 넣게되면 맨 앞에 0을 컴퓨터가 인식했을 때 8진수로 인식하게 된다.
int z = 0x10 이런식으로 넣었을 때는 16진수로 인식하게 된다.


기호 상수: 기호를 이용하여 상수를 표현한 것이다.
예) #define Exchange_rate 1120
    int won1 = Exchange_rate * dollar1;
    int won2 = Exchange_rate * dollar2;
생성 방법
1. #define 이용하기
2. const를 붙이기 constant의 약자로 변함없다는 것이다

실수를 표현하는 방법:
    1.고정 소수점 방식
    16비트 정수, 소수부분 16비트
    2.부동 소수점 방식
    부호비트 1비트  가수부분 23비트 , 지수부분 8비트
    10^-38~ 10^+38

float :32비트  1.17~*10^-38 ~ +- 3.40*10^38
double: 64비트 2.22*10^-308 ~ +- 1.79 * 10^308
